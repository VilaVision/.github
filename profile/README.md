## 🚀 **Quick Start**

<div align="center">

**Get up and running in 3 simple steps**

</div>

### 📋 **Prerequisites**
- Python 3.8+ • Git • Docker (optional)

### ⚡ **Installation**

```bash
# 1️⃣ Clone any repository
git clone https://github.com/VilaVision/<project-name>.git
cd <project-name>

# 2️⃣ Install dependencies
pip install -r requirements.txt

# 3️⃣ Run the project
python main.py
```
